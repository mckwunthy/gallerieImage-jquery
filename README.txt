Description : gallerie image interactive.
 
1. affichage d'image :
	--> Affichage des images dans une galerie

2. Naviguation interactive :
	--> Naviguer entre les images à l'aide de boutons de contrôle.

3. Zoom et rotation :
	--> Zoomer sur les images et faire pivoter pour afficher les détails.

4. Diaporama automatique : 
	--> Ajouter des filtres pour afficher les tâches accomplies ou non accomplies.

5. Diaporama automatique :
	--> Faire défiler les images automatiquement.
